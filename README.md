### Prerequisites

### Installation

1. Clone the repo
   ```sh
   git clone https://jovanjoto/Meiji-PropertyListing
   ```
2. Install Python dependencies
   ```sh
   pip install -r requirements.txt
   ```
2. Install NPM packages
   ```sh
   cd boundary
   ```
   ```sh
   npm install
   ```
4. Starting the frontend
   ```sh
   npm run dev
   ```
5. Starting the backend
   ```sh
   cd ..
   ```
   ```sh
   cd controller
   ```
   ```sh
   python application.py
   ```
